More Details On My Projects

Udemy Clone

In this project, we aimed to do an online learning system in which students can learn different technologies in a cheap and sustainable way. I did this project with my teammate Burak. We have used Node.js fr the backend part and and Html, CSS , Javascript for the frontend part. In this web application users who have registered can buy courses, watch videos, ask questions, comment on the videos and report the video if the content has inappropriate things. In the system teachers can upload courses and reply student's questions. Also, teachers can get notifications whenever new question or comment comes in. There is also anohter actor Admin in which s/he can ban students, teachers or courses due to incoming reports.

E-Voting

In this project, we aimed to do an online voting system in which people can vote for different topics and decide what to do. We have used Django which is a framework using Python. People can register and vote , as well as they can see the statistics of the voting after the voting is done. There is also another actor in the system which is premium user. Their advantage is, they can see statistics in more details.
